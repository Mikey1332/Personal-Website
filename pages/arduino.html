Infrared Remote

The Arduino Infrared Remote is a great way to control devices wirelessly. My fellow classmates created an arduino vehicle but didn't have a pratical way of controlling it. That's why I made the vehicle compatible with the Infarred Remote. The car detects the infrared signal it receives from the remote, and responds in accordance with the signal. For example, if it receives a “forward-signal” from the arduino remote, it will move forwards. The car can interpret signals for all of the following: Power, Forward, Backwards, Right, Left, Buzzer.

Display

A 7-segment display can show what the car's intentions are as it’s driving. If the car receives a signal to move forwards, the display will show an arrow pointing forwards to. There is a display for every function of the car.

Syncing

Now with the sync animation that runs when the car boots on, you can now sync any infrared remote to the car. For example, the car will show the logo for the power button in a blinking animation on the 7-segment display. When this animation is running, the user can press a button on any infrared remote and the car will save that signal and run the power function whenever it receives that signal. This happens for every function on the car.
